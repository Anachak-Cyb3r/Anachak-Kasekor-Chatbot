## [1.0.0] - 2025-01-12

### Added
- Initial release of Anachak Kasekor Chatbot
- Multi-language support (Khmer and English)
- Main bot hub with 6 specialized services:
  - Soil Type Detection Bot
  - Rice Seed Detection Bot
  - Rice Disease Detection Bot
  - Weather Forecasting Bot
  - Marketplace Bot
  - AI Agricultural Advisor Chatbot
- Interactive button-based navigation
- Language switching functionality
- Welcome image with compressed loading
- Environment variable configuration
- Comprehensive documentation
